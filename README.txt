因为SCAPI长期不更新，我(把红色赋予黑海)铲除了一部分屎山代码做出了SCEVO（中文名SCAPI进阶版）
SCAPIEVO简称SCEVO，更新频率更快，
群文件可以下载
每次主分支更新后，SCEVO出9个大版本之后就停更，催更主分支，1是防止差距过大，2是防止过于依赖EVO
SCEVO可以直接覆盖SCAPI1.50
以下为SCEVO的特点
1.修改窗口大小为原先80%（根据长度，不是体积）
2. .net版本更新为4.8.1（原4.7.2）
3.更新的第三方类库
4.删除无用文件
5.添加opencl，可以由模组调用
6.修复颜色格式，在HDR显示器上不会泛白了
7.锁定鼠标指针，防止点到悬浮窗
